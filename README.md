1. 운전면허 도로주행 시뮬레이터 (BestDriver_GENISIS)

C++ 기반 2D 운전면허 도로주행/주차 시뮬레이터 프로젝트입니다.
실제 자동차 면허 시험 규칙을 반영하여 A코스 시내 도로 주행, B코스 복합 도로 주행, T자 주차, 평행 주차를 구현하였고, 
신호 위반·중앙선 침범·역주행·안전벨트 미착용 등 시험 감점 항목을 판정할 수 있도록 구성하였습니다. 
또한 주행 결과를 점수판, 상세 결과창, 리플레이 화면으로 확인할 수 있도록 구현하여, 
사용자가 자신의 주행 실수를 직관적으로 파악할 수 있도록 설계하였습니다.

2. 개발 환경
 C++, SFML, ImGui, Visual Studio

 주요 기능
-시작 화면 및 메인 UI
  시간대에 따라 배경색이 변하는 시작 화면
  스크롤 및 애니메이션이 포함된 메인 진입 화면 구현
  
-차량 조작 시스템
  키보드 입력 기반 가속/감속/조향/기어 변경
  P/N/D/R 상태 표시 및 차량 물리 움직임 반영
  
-주행 코스 시뮬레이션
  A코스 시내 도로 주행 구현
  B코스 복합 도로 주행 및 회전교차로 포함 맵 구성
  
-2종 주차 모드
  T자 주차 
  평행 주차
  주차 슬롯 판정 및 성공 시각 효과 제공
  
-신호 및 위반 판정
  신호 위반, 중앙선 침범, 역주행, 안전벨트 미착용 판정
  실격 항목과 일반 감점 항목을 분리하여 처리
  
-리플레이 및 결과 확인
  주행 종료 후 리플레이 제공
  결과 점수판, 상세 결과창, 충돌 상세창 구현
  감점 내역과 발생 시간 확인 가능

3. 내가 맡은 역할
   
  시뮬레이션 완료 후 결과 점수판 및 메뉴 제작
  결과 판정 시스템 보완
  결과 데이터 생성 및 전달 흐름 연결 
  결과 상세창 및 충돌 상세창 구현
  결과 화면 한글화 및 UI 정렬/배치 개선
